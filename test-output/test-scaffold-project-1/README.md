# test-scaffold-project-1

A vue3 + Vite + typescript project.

## Tech Stack

- Vue 3
- Vite
- TypeScript





## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run tests
npm run test

# Lint code
npm run lint

# Format code
npm run format
```
