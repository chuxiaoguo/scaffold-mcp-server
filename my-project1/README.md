# my-project1

基于 vue3 + typescript 的项目。

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建项目
npm run build
```

## 项目结构

```
my-project1/
├── src/           # 源代码目录
├── dist/          # 构建输出目录
├── package.json   # 项目配置
└── README.md      # 项目说明
```

## 技术栈

- **框架**: vue3
- **语言**: typescript
- **路由**: vue-router
- **状态管理**: pinia
