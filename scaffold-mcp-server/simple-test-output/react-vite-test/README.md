# react-vite-test

A react + Vite + typescript project.

## Tech Stack

- React
- Vite
- TypeScript





## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run tests
npm run test

# Lint code
npm run lint

# Format code
npm run format
```
